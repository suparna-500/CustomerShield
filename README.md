# CustomerShield
CustomerShield is a customer churn prediction project where I built a machine learning model using the Telco dataset to identify customers at risk of leaving. The goal is to analyze behavior patterns and support businesses in improving customer retention.

CustomerShield is a customer churn prediction project where I plan to build a machine learning model to identify customers who are likely to leave a telecom service.

The goal is to analyze customer behavior patterns and help businesses make smarter retention decisions.

This project will include:

Data analysis and preprocessing

Model training using LightGBM

Model evaluation

Deployment using Streamlit
